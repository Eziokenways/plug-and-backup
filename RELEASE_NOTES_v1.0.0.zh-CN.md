# 随插备份 / Plug & Backup v1.0.0

这是随插备份 / Plug & Backup 的第一个公开版本。

随插备份是一个 Windows 便携托盘工具，用来自动备份 U 盘、读卡器、相机内存卡和外接移动硬盘中的照片、视频和相机 RAW 文件。程序不需要安装，下载 `PlugAndBackup.exe` 后直接运行即可。

## 主要功能

- 自动检测 USB 设备、读卡器、相机内存卡和外接移动硬盘。
- 支持照片、视频和常见相机 RAW 格式。
- 只复制文件，不删除 USB 或内存卡里的原文件。
- 默认不向内存卡或 USB 设备写入隐藏标记文件。
- 使用本机保存的复合指纹识别已信任来源。
- 支持“自动备份”和“插入后先确认再备份”两种模式。
- 支持“永久跳过”不需要备份的设备，避免反复弹窗。
- 按日期归档，也可选按照片 / 视频 / RAW、按设备、保留原设备目录结构分目录。
- 同名不同设备会自动追加短指纹后缀。
- 使用 SHA-256 内容哈希去重，避免重复备份。
- 同名不同内容不会覆盖，会自动改名保留新副本。
- 提供设置窗口、备份历史、日志文件夹、诊断信息导出和清理/卸载数据功能。

## 使用说明

首次运行时，程序会要求选择备份文件夹，并打开设置窗口。之后程序会常驻系统托盘。

右键托盘图标可以立即扫描、暂停自动备份、打开备份文件夹、打开最近备份位置、打开设置或退出程序。

## 注意事项

- 这是便携版程序，不需要传统安装器。
- 运行时不需要额外安装软件。
- 1.0 版本尚未做代码签名，首次运行时 Windows SmartScreen 可能会提示风险，这是未签名软件的常见现象。
- 程序不会删除 USB、内存卡或备份目录里的文件。
- 当前版本使用文件修改时间归档，暂不读取 EXIF 或视频拍摄时间元数据。
- MTP 手机如果没有 Windows 盘符，当前版本不会扫描。
