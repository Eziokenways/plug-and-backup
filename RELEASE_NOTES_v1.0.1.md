# Plug & Backup / 随插备份 v1.0.1

## 中文

这是一个小版本更新，主要优化 NAS 或网络备份目录离线时的体验。

### 更新内容

- 已设置过备份目录后，如果 NAS 或网络路径暂时离线，软件启动时不再强制弹出重新选择文件夹。
- 只有在检测到需要备份的 USB、内存卡或外接设备时，才提示备份目录不可用。
- 提示时可以选择“暂不备份”，或“重新选择备份位置”后继续本次备份。
- 发布用 exe 文件名已加入版本号：`PlugAndBackup-v1.0.1.exe`。

## English

This is a small maintenance update focused on the experience when a NAS or network backup folder is offline.

### Changes

- After a backup folder has already been configured, the app no longer forces folder selection on startup when a NAS or network path is temporarily offline.
- The unavailable-folder prompt now appears only when a USB drive, memory card, or external device is actually ready to back up.
- The prompt lets you skip backup for the current insertion or choose another backup folder and continue.
- The release exe filename now includes the version number: `PlugAndBackup-v1.0.1.exe`.
